# 📊 Exploratory Data Analysis of Global Economic Indicators

## 🌍Overview

This project explores global economic indicators using SQL-based exploratory data analysis techniques. The dataset contains country-level economic metrics including GDP, Population, Inflation, Unemployment, Exports, and Imports.

The goal of this analysis is to uncover trends, relationships, and insights that can support economic understanding and data-driven decision-making.

---

## 🎯 Objectives

- Analyze economic indicators across countries
- Identify top-performing economies
- Explore relationships between economic variables
- Generate actionable insights through SQL analysis
- Create visualizations to support findings

---

## 📂 Dataset Features

| Feature | Description |
|----------|-------------|
| Country | Country Name |
| GDP | Gross Domestic Product |
| Population | Population Size |
| Inflation | Inflation Rate (%) |
| Unemployment | Unemployment Rate (%) |
| Exports | Export Value |
| Imports | Import Value |
| Year | Reporting Year |
| GDP_Per_Capita | GDP divided by Population |

## 💡 Key Insights

- Large economies dominate global GDP rankings.
- Population size does not always correlate with economic output.
- Export-oriented countries tend to show stronger economic performance.
- GDP Per Capita provides deeper insight into prosperity levels.
- Economic indicators reveal significant variation across countries.
